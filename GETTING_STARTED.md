# Getting Started — For Non-Technical Business Founders

You don't need to be a developer to use any of these resources. This page tells you exactly where to start based on what you're trying to achieve.

---

## What Kind of Problem Are You Solving?

### "I want to win more customers without hiring more people."

**→ Start here: [AI GTM Playbook](./ai-gtm-playbook/)**

24 copy-paste AI prompt templates across the full sales funnel. No technical setup required — just open Claude or ChatGPT, paste a prompt, and follow the instructions. Each workflow shows you exactly what to type and what output to expect.

**First workflow to try:** [19 — ICP Builder](./ai-gtm-playbook/workflows/19-icp-builder.md) — define your ideal customer profile in 30 minutes.

## How These Guides Are Structured

Every guide in this repository follows the same format:

```
guide-name/
├── README.md          ← Start here. Overview, quick start, table of contents.
├── guides/            ← Step-by-step how-to guides (numbered 01, 02, 03...)
├── templates/         ← Copy-paste templates (emails, proposals, checklists)
└── scripts/           ← Optional automation scripts (for technical users only)
```

**You only need to read the `guides/` folder.** Templates are extras. Scripts are optional.

## Tools You'll Need

Most guides in this repo require only:

| Tool | Cost | What It's For |
|------|------|---------------|
| Claude (claude.ai) | Free / $20/month | AI prompts and writing |
| ChatGPT (chat.openai.com) | Free / $20/month | Alternative to Claude |
| Google Docs / Notion | Free | Storing outputs and templates |

You do **not** need to know how to code, run a server, or use a terminal for the majority of these guides.

## What If I Get Stuck?

1. **Read the FAQ** in the relevant guide folder — most common questions are answered there.
2. **Open an issue** in this GitHub repo — describe what you're trying to do and where you got stuck.
3. **Ask in your EO chapter** — these guides were built by and for EO members.

## A Note on Scripts and Technical Guides

Some guides include optional scripts (Python, Bash, PowerShell). These are clearly marked and always have a **non-technical alternative** listed alongside them.

If you see a code block and feel lost, scroll past it — there is always a manual or simpler option described nearby.
