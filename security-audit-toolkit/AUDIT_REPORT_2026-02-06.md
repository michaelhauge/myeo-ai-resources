## Remaining Action Items

### Priority 1: Enable Firewall (5 minutes)
1. Open **System Settings**
2. Go to **Network** → **Firewall**
3. Click **Enable Firewall**
4. Enable **Stealth Mode** (optional but recommended)

### Priority 2: Secure SSH Access (2 minutes)
**Option A:** Disable if not needed
1. Open **System Settings**
2. Go to **General** → **Sharing**
3. Turn off **Remote Login**

**Option B:** Restrict to VPN only
- Keep Remote Login on but only access via Tailscale/VPN
- Firewall will help block external access

### Priority 3: Clean Shell History (10 minutes)
1. Edit `~/.zsh_history` manually
2. Search for and remove lines containing API keys
3. Consider using: `export HISTIGNORE="*sk-*:*AKIA*:*ghp_*"` in ~/.zshrc
